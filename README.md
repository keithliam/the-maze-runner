# The Maze Runner

A simple maze puzzle equipped with a heuristic AI.

## Objective

To beat the game, you need to move the player around to get to the goal.

## Game Controls

* <kbd>↑</kbd> move up
* <kbd>↓</kbd> move down
* <kbd>←</kbd> move left
* <kbd>→</kbd> move right

## How to run


```console
$ ./main
```
